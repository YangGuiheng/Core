一、概述

1.1 定义

Core Industrial Sector（CIS）是 Core 的工业制造与资源采集体系，负责 Core 生态内所有硬件产品的原材料采集、制造与维护。CIS 采用三元结构，覆盖从资源采集到重型机械制造、再到微观电子制造的全产业链条。

1.2 设计原则

原则	说明
采制一体	CISC 采集原料，CISH/CISL 制造产品，形成完整工业闭环
分布式节点	各区域独立部署制造节点，产品版本与产地绑定
区域协同	同一区域内 CISH 与 CISL 协同完成整机装配
全链可控	从原材料到终端产品，Core 掌握完整制造链条
1.3 工业体系架构

text
Core Industrial Sector（CIS）
   │
   ├── CISC（资源采集）
   │   ├── 月球矿产（氦-3、钛、铁、铝、稀土）
   │   ├── 小行星矿产（铂族金属、稀有元素、水冰）
   │   ├── 气态巨行星采集（氢气）
   │   └── 地球本地矿产（基础原料）
   │
   ├── CISH（重工业）
   │   ├── Core Rocket（CR）制造
   │   ├── Core Star（CS）制造
   │   ├── CSN 节点制造
   │   ├── CSS 舱段制造
   │   └── CIN 结构制造
   │
   └── CISL（轻工业）
       ├── CAPU 制造
       ├── CGPU 制造
       ├── CNBP 制造
       ├── 芯片制造
       └── 电子模块制造
二、CISC：资源采集

2.1 定位

CISC 是 Core 工业体系的原料供应部门，负责为 CISH（重型制造）和 CISL（轻型制造）提供原材料，同时为 CEC（能源体系）提供核聚变燃料。

2.2 采集范围

采集类型	来源	主要用途
氦-3	月球表面	核聚变燃料
钛、铁、铝、稀土	月球矿产	CISH 重型制造原料
铂族金属、稀有元素	小行星带	CISL 电子制造原料
水冰	月球/火星/小行星	燃料、生命支持
氢气	木星大气	核聚变燃料
基础矿产	地球本地	CISH/CISL 制造原料
2.3 与 CEC 的协同

CISC 采集的氦-3 和氢气直接供给 CEC 的核聚变发电设施，形成燃料闭环：

text
CISC 采集 → 燃料处理 → CEC 核聚变 → 电力供应 → CISH/CISL 制造
三、CISH：重工业

3.1 定位

CISH 是 Core 的重型机械制造部门，负责 Core 所需的大型硬件平台和结构件的制造与总装。

3.2 产品范围

产品	说明
Core Rocket（CR）	运载火箭制造与总装
Core Star（CS）	卫星平台制造
CSN 节点	通信网络节点硬件制造
CSS 舱段	空间站舱段制造
CIN 结构	综合节点结构制造
3.3 与 CAD 的协同

CISH 制造 Core Rocket，交付 CAD 执行发射
CISH 制造空间站舱段，由 CAD 负责轨道部署
CISH 提供轨道资产结构件，CAD 负责日常运维
四、CISL：轻工业

4.1 定位

CISL 是 Core 的微观电子制造部门，负责 Core 所需的所有电子设备、芯片和终端产品的制造。

4.2 产品范围

产品	说明
CAPU	Core APU 玄央处理器
CGPU	Core 通用处理板
CNBP	Core 笔记本 Pro
芯片	Core 生态内各类芯片
电子模块	终端设备、节点控制器等
4.3 与 CISH 的协同

CISL 制造电子模块和控制系统，集成到 CISH 制造的重型机械中
同一区域的 CISH 和 CISL 协同完成整机装配
五、分布式节点

5.1 部署原则

CISH 和 CISL 均采用全球分布式节点部署，产品版本与产地绑定。

区域	版本标识	负责产品
Asia	A版	CNBP-A、CGPU-A、CAPU-A
Europe	E版	CNBP-E、CGPU-E、CAPU-E
North America	N版	CNBP-N、CGPU-N、CAPU-N
其他区域	对应标识	对应产品版本
5.2 优势

优势	说明
供应韧性	单一节点受损，其他节点可继续生产
物流优化	区域生产、区域供应，减少长途运输
技术扩散控制	即使一个区域的技术被外流或限制，整体架构不受影响
产品多样性	不同版本可针对不同区域需求微调
六、工业体系流程

6.1 完整工业链

text
CISC（采集）
   │
   ├── 矿石/金属 → CISH（重型制造）
   │   ├── CR、CSN、CSS、CIN、CS
   │   └── 交付 CAD（发射与运营）
   │
   ├── 稀土/硅 → CISL（轻型制造）
   │   ├── CAPU、CGPU、CNBP
   │   └── 进入 Core 产品分发渠道
   │
   └── 氦-3/氢气 → CEC（能源）
       └── 电力供应 → CISH/CISL 制造
6.2 部门协作关系

协作方	内容
CAD（航空部门）	CISH 制造 Core Rocket 和轨道平台，CAD 负责发射与轨道运营
CEC/CEP（能源体系）	CISC 为 CEC 提供核聚变燃料，CEC 为 CISH/CISL 提供电力
CJMT（维护团队）	CISH 提供备件，CISL 提供电子模块，共同支持 CJMT 维护任务
Core 产品分发	CISL 制造的消费电子产品进入 Core Store 或直接销售渠道
