一、概述

1.1 定义

Core 社会体系是 Core 生态内部的社会契约与保障框架，涵盖基本社会保障、机会公平、职业尊严与社会参与等维度。它是 Core 经济体系的补充，确保经济活动的成果惠及所有人，而不是仅服务于贡献者或高收入群体。

1.2 设计原则

原则	说明
人人有保障	无论贡献水平如何，每个人都有权获得基本生存保障
机会公平	社会流动性不受出身、职业或初始条件的限制
劳动尊严	所有劳动都被认可，无论其CV效率高低
社会参与	每个人都有参与Core社会生活的权利
归属感	Core 不仅是技术体系，也是人们可以归属的社区
二、Core Insurance（CI）

2.1 定义

CI 是 Core 的基本社会保障机制，为低 CV 且低 SE 的个人提供基本生存保障与再融入通道。它不是福利，不是施舍，而是基本生存权的保障。

2.2 适用对象

条件	说明
持有有效 CID	必须是 Core 生态内注册的个人
CV 低于阈值	低于 Core Bank 设定的最低贡献标准
SE 低于阈值	低于 Core Bank 设定的最低资产标准
主动申请	需要个人主动申请，非自动发放
2.3 CI 的覆盖范围

保障内容	说明
基本食物	保障基本营养需求
基本住所	保障基本居住条件
基本医疗	覆盖在 Free Medical 体系中
基础生活设施	水电、通信等基本生活服务
2.4 CI 的核心逻辑

text
低CV低SE → 进入CI → 获得基本保障 + 技能培训 → 进入劳动体系 → 开始获取CV → CV/SE达标 → 退出CI
CI 不是“养懒人”，而是“接住人，再送出去”。它有进入门槛，也有退出条件；有保障，也有方向；有培训，也有对接。

2.5 退出机制

退出条件	说明
CV 达标	CV 累计达到或超过设定阈值
SE 达标	SE 累计达到或超过设定阈值
获得稳定工作	进入 CIS、CAD、CEC 或其他 Core 部门的劳动体系
自动退出	当 CV 和 SE 同时超过阈值时，自动退出 CI
主动退出	个人可主动申请退出 CI
三、Free Medical

3.1 定义

Free Medical 是 Core 的基本医疗保障体系，覆盖所有持有有效 CID 的个人。它不设门槛、不与 CV/SE 挂钩，确保健康权不成为经济能力的附属品。

3.2 覆盖范围

类别	内容
基础医疗	常见病诊疗、预防性检查、基础药物
急救	急诊处理、抢救、紧急手术
慢性病管理	长期疾病的基础治疗、定期随访
妇幼保健	产检、分娩、儿童基础保健
心理健康	基础心理健康支持
康复	基础康复治疗
3.3 不覆盖范围

类别	处理方式
非必需的整容/美容手术	自费（SE支付）
实验性治疗	需要特别审批
高成本特殊治疗	可能需要 SE 支付或补充保险
非紧急的长途医疗转运	视情况评估
3.4 Free Medical 的哲学基础

健康是基本人权，不是 CV 或 SE 的兑换品
一个人不应该因为“贡献不够”而无法获得基础医疗服务
Free Medical 与 Core 的 CI 体系一致：保证最低生存与健康标准
四、技能培训与社会流动

4.1 定义

技能培训是 CI 的重要组成部分，也是 Core 社会流动性的核心通道。它为低 CV 低 SE 的人提供提升自身能力、进入劳动体系的机会。

4.2 培训层级

层级	内容	对应方向
基础	通用技能（沟通、协作、基础工具使用）	任何方向的基础能力
专项	CIS操作、CAD辅助、CEC设备维护、物流管理、农业/生态维护	CISH/CISL生产、CAD地面支持、CEC设施运行、供应链与运输、生态与生活支持
高级	节点维护、轨道器操作、能源系统管理	特定岗位资格认证
每个层级都与 Core 的部门需求对接，培训不是泛泛而学，而是有目标岗位的定向培训。

4.3 培训期间的 CV 获取

阶段	CV 获取方式
完成培训课程	获得一次性 CV 奖励
通过技能认证	获得额外 CV 奖励
进入劳动体系初期	享受“过渡期加成”，加速 CV 积累
这确保 CI 领取者不会长期停留在低 CV 状态，而是有明确的路径逐步向上移动。

4.4 社会流动的承诺

承诺	说明
不因出身设限	任何 CID 持有者都可以参加培训
不因职业定型	职业路径可以转换，CV 积累方式可变
不因初始条件停滞	CI 提供起点，但不是终点
技能认可制度	培训完成后获得 Core 认可的技能认证，可在不同部门间转移
五、职业尊严与劳动价值

5.1 核心原则

Core 不认为人有高低贵贱之分，也不认为职业与 CV 挂钩。

一名环卫工的 CV 值可能高于一个 Office Worker
职业不决定 CV 上限，没有任何职业被预设为“低 CV 职业”
所有劳动都可转化为 CV，只要它对 Core 的运行有贡献
CV 不是对人的分层工具，而是对贡献行为的记录
5.2 CV 获取的平等性

原则	说明
任何人都可以以相对简单的方式获取 CV	入门门槛低
单位时间内获取的 CV 数与获取难度成正比	高技术行为单位时间 CV 获取效率更高
熟练工人与新工人 CV 获取速率不同	熟练工人在同一岗位单位时间内获取的 CV 高于新工人
5.3 劳动尊严的表达

Core 公开承认体力劳动的价值，其重要性在公开场合与高技术劳动并列对待
CI 和 Free Medical 的存在本身就是一种态度的表达
技能培训确保所有劳动者都有向上流动的可能
六、社会参与与归属感

6.1 定义

Core 不仅是一个技术体系，也是一个社区。社会参与和归属感是 Core 社会体系的重要组成部分。

6.2 社会参与渠道

渠道	说明
社区投票	所有持有 CID 的个人均可参与 Core 治理投票
社区提名	社区成员可以提名他人获得 CV 认可
地方社区	按区域、职业、兴趣划分的地方社区
公共活动	Core 举办的公共活动、会议、庆典
反馈机制	向 Core 基金会反馈意见与建议的正式渠道
6.3 归属感的文化表达

Core 的旗帜、徽章、标识由社区成员共同设计
年度 Core 日纪念活动
Core 历史记录与档案向所有成员开放查阅
新成员入籍仪式（可选项）
6.4 社区提名与认可

社区成员可提名其他成员获得 CV（表彰未被系统自动记录的贡献）
提名需经过社区委员会审核
被认可的贡献类型包括：社区建设、互助行为、长期支持、隐性贡献等
提名成功者获得 CV 奖励，提名者获得少量 CV 作为感谢
七、社会与经济的接口

社会机制	经济接口
CI	提供基本保障，降低贫困对经济活动的阻碍
Free Medical	保障健康，减少因病致贫
技能培训	提升劳动能力，增加 CV 获取效率
社区提名	补充 CV 体系的覆盖盲区
社会参与	增强成员对 Core 的认同感和长期投入意愿
Core 的社会体系不是经济的附属品，而是经济的平行结构。它确保经济活动服务于人，而非人服务于经济活动。它提供安全网、提供通道、提供归属感——让 Core 不仅是一个可以工作的地方，也是一个可以生活的地方。
