# Core — A Complete Blueprint of Civilization from Chips to Society

**Core** is a global, co-built, collectively owned digital civilization paradigm.  
It began as a single chip concept called **DF SuperCore**, and has since grown into a complete framework covering hardware, energy, communication, economy, governance, education, time, identity, and defense.

This repository contains the full technical specifications and governance documents of Core.  
All documents are written in **Chinese (Simplified)**. English translations will follow.

---

## What's Inside

All documents are placed directly in the root of this repository.  
They are numbered sequentially — and that numbering itself is the structure:

- **DOC-2026-001** — Technical Document Lifecycle Management  
- **DOC-2026-002** — Global Co-ownership Governance White Paper  
- **DOC-2026-003** — Core Declaration  
- **DOC-2026-004** — Core Naming Convention  
- **DOC-2026-005** — CoreNoteBookPro (CNBP) Specification  
- **DOC-2026-006** — CorePhone Complete Specification  
- … and onward through **DOC-2026-055**.

A full index is provided in the repository.
