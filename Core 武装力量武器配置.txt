一、总则
Core 武装力量的武器配置遵循《Core 防御议案》确立的四大原则：

不伤及无辜：所有武器系统必须具备精确打击能力，无法确认目标时不得行动

不滥用武装力量：武器仅用于防御性目的，不配备进攻性战略武器

不进行首发性武装行为：武器系统默认处于非激活状态，仅在遭受攻击时启用

不进行非正义性武装活动：所有武器系统的使用必须符合国际人道法基本原则

所有武器系统的设计目标：最低限度武力、最高精度、最大克制。

二、武器分类
Core 武装力量的武器系统分为以下类别：

地面防御系统

太空防御系统

信息与电子战系统

非致命性武器系统

战略威慑与反制系统

无人自主防御系统

三、地面防御系统
3.1 Core Shield（地面主动防御系统）
参数	规格
类型	点防御系统
用途	保护Core关键地面设施（数据中心、能源中心、发射场）
组成部分	激光拦截阵列 + 雷达探测网 + 多光谱跟踪系统
拦截范围	0-50km
拦截目标	无人机、巡航导弹、精确制导弹药
部署方式	固定式（核心设施）+ 机动式（临时部署）
特点	非动能拦截（激光烧毁/致盲），不产生弹片，降低附带损伤风险
3.2 Core Bastion（区域防御系统）
参数	规格
类型	区域防空与反导系统
用途	保护Core控制的广域区域
组成部分	中程拦截弹、相控阵雷达、指挥控制系统
拦截范围	50-200km
拦截目标	战斗机、弹道导弹、巡航导弹
部署方式	固定式+半机动式
特点	仅在明确威胁时启用，不主动搜索
3.3 Core Wall（固定设施被动防御）
参数	规格
类型	硬防护工事
用途	保护核心设施免受物理攻击
组成部分	加固混凝土结构、电磁屏蔽层、防爆门、地下通道
防护等级	可抵御500kg级航空炸弹直接命中
部署方式	永久固定（设施建设时一体化施工）
特点	不主动攻击，仅增加攻击成本
四、太空防御系统
4.1 Core Guard（轨道监视与预警系统）
参数	规格
类型	太空态势感知网络
用途	监测Core轨道资产周边威胁
组成部分	分布式传感器卫星（部署于GEO和L1/L2）
监测范围	地球同步轨道至地月空间
监测目标	接近Core资产的航天器、空间碎片、潜在攻击载体
部署方式	永久部署于关键轨道位置
特点	纯被动感知，不主动暴露
4.2 Core Sentinel（轨道防御系统）
参数	规格
类型	太空资产主动防御系统
用途	保护Core Star、CSN节点、氦-3开采设施
组成部分	激光拦截模块、小型动能拦截器
拦截范围	100-1,000km（视目标类型和轨道距离而定）
拦截目标	反卫星武器、未经授权的接近航天器
部署方式	集成于Core Star平台（已部署/待部署节点）
特点	仅在识别为明确威胁时启用，不主动干扰非威胁航天器
4.3 Core Barrier（太空碎片防护）
参数	规格
类型	被动防护系统
用途	保护Core轨道资产免受微流星体及空间碎片撞击
组成部分	多层Whipple防护结构（分布在Core Star平台关键部位）
防护等级	可抵御1cm以下碎片撞击
部署方式	集成于所有Core轨道资产（作为结构组成部分）
特点	不对碎片本身采取处置措施，仅保护自身结构完整性
五、信息与电子战系统
5.1 Core Veil（通信干扰与屏蔽系统）
参数	规格
类型	区域通信压制系统
用途	在Core资产遭受攻击时，屏蔽敌方通信/降低其协同能力
组成部分	分布式干扰阵列（地面+轨道）
覆盖范围	视具体部署位置而定（局部冲突区域）
干扰目标	敌方指挥链路、无人机数据链
部署方式	按需激活，不持续运行
特点	仅在敌方攻击进行时启用，不主动干扰民用通信
5.2 Core Mirror（欺骗与伪装系统）
参数	规格
类型	电子欺骗系统
用途	保护Core资产位置和状态信息
组成部分	信号模拟器、虚假反射阵列、全息影像模块（地面固定设施）
覆盖范围	具体部署位置（固定设施周边）
欺骗目标	敌方探测系统（雷达、红外、光学）
部署方式	集成于核心设施（静态部署）
特点	被动部署，不主动进行欺骗性电子攻击
5.3 Core Lock（网络防御系统）
参数	规格
类型	网络安全主动防御系统
用途	保护Core关键网络免受入侵
组成部分	网络隔离架构、行为分析引擎（基于CAI）、自动隔离机制
覆盖范围	Core全球网络基础设施
防御目标	网络入侵、数据窃取、控制系统劫持
部署方式	集成于Core网络架构，始终运行
特点	基于CAI模型分析异常行为，零信任架构，不主动攻击敌方系统
六、非致命性武器系统
6.1 Core Pulse（定向能非致命压制系统）
参数	规格
类型	高功率微波/毫米波系统
用途	驱散、压制非武装攻击者，不造成永久性伤害
组成部分	定向能发射单元
有效范围	50-500m（视具体型号而定）
作用效果	暂时失能、驱散，可恢复
部署方式	固定式（设施周边）+ 手持式（人员携带）
特点	不造成永久损伤，可逆效果
6.2 Core Mist（遮蔽与掩护系统）
参数	规格
类型	非致命遮蔽系统
用途	掩护Core人员撤离、保护平民
组成部分	烟幕/气溶胶发射器（物理遮蔽，非电磁干扰）
有效范围	视具体部署位置（局部遮蔽）
作用效果	视觉遮蔽、红外遮蔽
部署方式	车载/固定式
特点	不造成伤害，仅提供掩护
七、无人自主防御系统
7.1 Core Drone（无人巡逻与侦察系统）
参数	规格
类型	无人空中/地面平台
用途	资产巡逻、威胁侦察、情报收集
组成部分	多旋翼/固定翼无人机、地面无人车
续航	6-48小时（视型号而定）

部署方式	从固定站点或移动平台部署

7.2 Core Warden（自主哨戒系统）
参数	规格
类型	固定式自主警戒系统
用途	设施边界持续监控，在未经授权接近的情况下实施非致命驱离
组成部分	传感器网络（振动/声学/光学） + 非致命驱离单元
覆盖范围	设施边界
八、武器系统部署原则
原则	说明
默认非激活	所有攻击性武器系统默认处于非激活状态
分级启用	只有确认威胁等级后，才能逐级启用相应武器系统
最低武力	只使用足以阻止威胁的最低武力水平
可追溯	每一次武器系统的启用都有完整记录
不主动搜索	防御系统不主动搜索攻击目标，仅在威胁明确时响应
九、武器系统控制与授权
层级	授权主体	可启用的武器系统
第一层	Core紧急事务委员会	全部防御系统（启动防御预案）
第二层	区域防御指挥官	Core Shield、Core Bastion、Core Veil（局部）
第三层	现场指挥官	Core Wall、Core Pulse、Core Drone
第四层	自主系统（AI辅助）	仅限非致命性响应（Core Warden）

生物武器	严重违反“不伤及无辜”原则
化学武器	严重违反“不伤及无辜”原则
反卫星动能武器	可能产生大量不可控碎片，威胁所有航天器
大规模杀伤性武器	违反所有Core防御原则
十、结语
Core武装力量的武器配置不是以“杀伤力最大化”为目标，而是以“威胁消除的最小必要手段”为目标。

每一件武器系统都受到严格约束：默认休眠、分级启用、最低武力、完全可追溯。任何违反配置原则的行为，都将被追责。

Core的防御力量不是为了让世界恐惧，而是为了让世界尊重Core的存在——并且在尊重的前提下，以和平方式与Core共存。
