一、总则
“我们是有剑不用，不是手里没剑。”

Core 的武装力量必须在构成上完整，在行动上克制。不能因为主张和平，就放弃对自身安全的绝对保障。步兵、装甲与无人化作战平台是 Core 武装力量在物理世界中维持防御与威慑的基石。

以下所有单位与装备，均严格遵循《Core 防御议案》四大原则：不伤及无辜、不滥用武装力量、不进行首发性武装行为、不进行非正义性武装活动。

二、步兵单位（Core Infantry）
2.1 定位与职能
职能	说明
设施防卫	负责核心设施（CEC、发射场、数据中心）的地面近距防卫
区域巡逻	在 Core 控制的区域执行巡逻与警戒任务
快速反应	在突发威胁发生时，第一时间抵达现场进行处置
人道主义保护	在冲突区域保护平民安全，提供安全撤离通道
2.2 装备配置
类别	装备	说明
防具	Core Shell（轻量化防弹插板+头盔）	提供对轻武器射击和破片的有限防护
单兵武器	Core Lance（模块化电磁步枪）	可发射非致命弹（驱散）与动能弹（精确打击）双模式，具备火控辅助瞄准系统
辅助设备	Core Eye（单兵增强现实观测/通信终端）	提供战场态势感知、导航与加密通信
非致命装备	Core Pulse（手持式非致命压制器）	用于驱散非武装攻击者（毫米波/声波压制）
运输载具	Core Transport（轻型全地形运输车，可选配武器站）	用于步兵快速部署，可加装非致命/轻武器站
2.3 作战原则
不主动出击，仅在 Core 设施或人员遭受直接威胁时介入

使用武力必须遵循 “最低限度” 原则

确认攻击目标前，必须排除平民存在的可能性

非致命选项优先，致命武力作为最后手段

三、装甲力量（Core Armor）
3.1 定位与职能
职能	说明
重装防卫	为核心设施提供重型防护，抵御装甲/重火力的直接攻击
机动防御	在 Core 控制区域外围建立机动防御屏障
区域拒止	在极端情况下，对武装攻击者的推进路线形成阻断
3.2 装备配置
型号	类型	配置
Core Bastion	重型步战车（轮式/履带式）	配备 30mm 电磁自动炮（可切换非致命/致命弹种）、烟雾/遮蔽发射器、复合装甲
Core Wall	轻型装甲巡逻车	用于日常巡逻与快速反应，配备 12.7mm 遥控武器站（非致命优先）
Core Shield	机动式主动防御系统载车	搭载激光拦截模块，用于伴随装甲单位提供主动防御，拦截反坦克导弹和火箭弹
3.3 作战原则
装甲力量仅用于防御性任务，不参与主动进攻作战

行动前必须确认目标区域内无平民

不得对民用目标或非武装目标使用主武器

在无法确认攻击来源时，不得进行盲目还击

四、无人化作战平台（Core Autonomous Force）
4.1 定位与职能
无人化作战平台是 Core 武装力量的重要组成部分，负责执行高危险、高精度或长航时的任务，最大限度减少人员伤亡。

职能	说明
前沿侦察	在步兵到达前，对威胁区域进行无人侦察与评估
低烈度对抗	在确认威胁目标后，执行精确打击或压制任务
持续巡逻	在 Core 控制区边界进行无人值守巡逻
后勤支援	承担弹药补给、伤员后送等任务
4.2 平台分类
A 类：空中无人作战平台

型号	类型	配置
Core Eagle	中高空长航时无人机	配备光电/红外侦察设备、激光指示器、精确制导弹药（可选）
Core Hawk	小型战术无人机	用于班排级侦察与目标定位，可挂载 2 枚轻型精确制导弹药
Core Wasp	蜂群无人机	用于饱和式侦察或电子干扰，可携带非致命载荷
B 类：地面无人作战平台

型号	类型	配置
Core Hound	中型无人战车	配备遥控武器站（7.62mm/12.7mm，可切换非致命/致命模式）
Core Ant	微型地面机器人	用于建筑物清场、隧道侦察，可携带非致命压制装置
Core Mule	无人后勤运输车	用于弹药补给、伤员后送与物资运输
C 类：自主哨戒系统

型号	类型	配置
Core Sentinel	固定式自主警戒系统	集成振动/声学/光学传感器，配备非致命驱离装置（声波/激光致眩）
4.3 无人化平台作战原则
所有无人化平台必须由人类操作员确认目标后方可开火（“人在回路”原则）

禁止自主开火模式，任何情况下不得由 AI 独立决定使用致命武力

无人平台不得用于主动攻击民用目标或非武装人员

无人平台在执行任务期间，必须持续广播“非攻击状态”信标，避免误判

五、组织与指挥
层级	单位	指挥权归属	启用条件
第一层	Core 紧急事务委员会	全局最高指挥权	防御预案启动时接管
第二层	区域防御指挥部	区域指挥官	威胁明确、区域级别响应
第三层	战术单位指挥官	现场指挥官	具体威胁处置
所有单位（含无人化平台）的启用与部署，均需符合“威胁确认-授权批准-行动执行-事后审查”的完整链条。

六、限制与约束
不配备大规模杀伤性武器（如核、生、化武器）

不主动部署于 Core 控制区之外

不参与任何形式的侵略性军事行动

不干涉他国内政

不用于对平民的镇压或控制

无人化平台不得拥有自主开火权

七、结语
Core 拥有步兵、装甲与无人化作战平台，是为了确保自身在物理世界中的存在不被轻易抹去。

“我们是有剑不用，不是手里没剑。”

剑的意义在于：当世界试图用暴力阻止 Core 的存在时，Core 有能力保护自己。但剑的归宿永远是刀鞘——Core 不会主动拔剑，除非别无选择。

More Core, More Possible.
For a world without masters.
