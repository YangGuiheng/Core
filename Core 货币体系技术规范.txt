一、概述

1.1 定义

Core 经济体系是 Core 生态内部的货币与金融基础设施，由 Contribution Value（CV） 和 Star Entropy（SE） 两种货币组成，分别承担贡献计量与流通交易的职能。该体系由 Core Bank 统一管理，不与任何外部主权货币挂钩，其价值锚定于 Core 内部的实际产出与物价水平。

1.2 设计原则

原则	说明
贡献驱动	CV 只能通过实际贡献获得，不可购买、不可转让
价值内生	SE 的价值由 Core 内部购买力决定，不依赖外部锚定
双轨并行	CV 与 SE 功能分离、互补，形成完整的经济闭环
市场规则	Core Bank 基于信用与市场原则运作，差异化服务不同信用等级
二、货币体系总览

维度	CV	SE
性质	贡献凭证	基础货币
获取方式	贡献行为	兑换、贸易、授信、劳动报酬
主要功能	衡量贡献、提升权限、信用凭证	购买商品、服务、支付、贸易结算
是否可流通	不可转让	自由流通
发行依据	贡献行为本身	Core 经济总量和储备
供应量控制	自然产生	Core Bank 调控
三、Contribution Value（CV）

3.1 定义

CV 是 Core 生态内衡量个人或组织贡献的唯一凭证。它不可购买、不可转让、不可交易，只能通过实际贡献行为获得。

3.2 获取方式

贡献类型	说明
代码/系统贡献	代码提交、系统维护、漏洞修复
工业制造	CISH/CISL/CISC 的生产与维护
能源供应	CEC/CEP 的能源生产与输送
轨道运维	CAD/CSS/CIN 的轨道资产维护
社区治理	参与投票、治理、文档编写
体力劳动	基建、物流、清洁、农业等
技能培训	参与 Core 认可的技能提升计划
其他贡献	任何对 Core 运行有实际帮助的行为
3.3 CV 获取速率

劳动类型	入门门槛	CV 获取速率（单位时间）
体力劳动（入门级）	极低	1x
体力劳动（熟练级）	低	1.5x - 2x
技能劳动（入门级）	中等	1.5x - 2x
技能劳动（熟练级）	中等	3x - 4x
高技术劳动（入门级）	高	3x - 5x
高技术劳动（熟练级）	高	6x - 10x
CV 获取速率与劳动难度和熟练度成正比，但所有劳动均可获得 CV。

3.4 CV 的消耗方向

消耗方向	说明
兑换 SE	按 100 CV = 1 SE 的初始比例兑换
CID 权限提升	一次性消耗 CV 提升权限等级
CID 权限维护	周期性消耗 CV 维持当前权限等级
临时权限	用 CV 换取限时权限提升
3.5 CID 权限提升消耗

权限等级	一次性升级消耗	维护费用（每年）
基础（5xx）	0	0
中级（6xx）	1000 CV	100 CV/年
高级（7xx）	5000 CV	300 CV/年
核心（7xx+）	20000 CV	1000 CV/年
决策层	50000 CV	5000 CV/年
维护费不缴纳则权限降级，已消耗 CV 不返还。

3.6 临时权限

临时权限类型	CV 消耗	时长	获得权限
轻度临时权限	100 CV	7天	+1 权限等级
中度临时权限	300 CV	30天	+1 权限等级
重度临时权限	500 CV	30天	+2 权限等级
关键任务临时权限	1000 CV	90天	+2 权限等级，附加特殊权限
临时权限不占用永久权限位置，到期后自动失效。

四、Star Entropy（SE）

4.1 定义

SE 是 Core 的基础流通货币，承担价值尺度、流通手段、支付手段和贮藏手段的职能。它是 Core 内部交易与对外经贸的结算单位。

4.2 发行机制

SE 由 Core Bank 统一发行
发行总量由 Core Bank 根据 Core 经济总量、物价水平和储备情况进行调控
SE 的发行与 CV 兑换、贸易结算、信贷投放等经济活动动态关联
SE 的供应量不与任何外部货币挂钩
4.3 获取方式

获取途径	说明
CV 兑换	按 100 CV = 1 SE 的初始比例兑换
外部货币兑换	按 Core Bank 公布的汇率兑换
劳动报酬	以 SE 形式支付的工资或服务费
贸易收入	通过 Core 内外部贸易获得
Core Bank 授信	基于 CV 信用获得的 SE 额度
4.4 SE 的用途

购买 Core 生态内的商品与服务
支付 Core 内外部贸易结算
作为 Core 对外经贸的结算货币
存储、投资、信贷
五、汇率机制

5.1 核心原则

SE 不与任何外部主权货币固定挂钩，其对外汇率由 Core 内部购买力平价（PPP） 决定。

5.2 价值锚定

SE 的价值锚定于 Core 内部的一篮子商品与服务，而非外部货币或单一商品。

5.3 汇率形成机制

Core Bank 定期（每季度）编制 Core 内部 CPI（居民消费价格指数）
CPI 篮子包含 Core 内部核心商品与服务的价格
以 1 SE 在 Core 内部的购买力为基准
将该购买力投射至外部市场同类商品/服务的价格
对外汇率由 Core Bank 测算并公布
5.4 操作示例

1 SE 在 Core 内部可购买 1 个 CorePhone 摄像头
该摄像头在目标国的当地价格为 X 元
则 1 SE 对目标国货币的汇率参考为 X 元
5.5 汇率调整机制

调整依据	说明
Core 内部 CPI 变动	物价上涨 → SE 购买力下降 → 对外汇率调整
外部市场对标商品价格变动	外部价格变化 → 反映在汇率测算中
周期性评估	Core Bank 定期（每季度）公布汇率参考值
5.6 对外汇率的作用

Core 对外经贸的结算汇率
外部货币兑换 SE 的汇率
Core Bank 外汇储备的价值参考
六、Core Bank

6.1 定位

Core Bank 是 Core 货币体系的管理与运营机构，负责 SE 发行、信贷管理、汇率管理与储备管理。它由 Core Foundation 直接管辖，是 Core 经济体系的核心基础设施。

6.2 Core Bank 的职能

职能	说明
SE 发行	根据 Core 经济总量调控 SE 发行
信用评估	基于 CV、SE、历史行为评估信用等级
信贷管理	提供授信额度、贷款、结算服务
汇率管理	测算并公布 SE 对外汇率
储备管理	管理 Core 的经济储备与外汇储备
货币政策	制定并执行 Core 内部货币政策
6.3 信贷差异化机制

信用等级	信贷条件
信用优异（高CV 或 高SE）	优先审批、低利率、长周期、高额度
信用良好（中等CV 或 中等SE）	标准审批、中等利率、一般周期、中等额度
信用一般（低CV 且 低SE）	排队审批、较高利率、短周期、低额度
信用不足（无CV 且 无SE）	高门槛审批、高利率、需要抵押
6.4 贷款用途限制

与 Core 生态建设相关的生产、贸易、服务活动
与 Core 对外经贸相关的结算与支付
与 Core 成员个人发展相关的技能培训、设备采购
贷款不得用于与 Core 无关的外部投机活动
6.5 储备管理

Core Bank 管理以下储备：

储备类型	说明
SE 储备	Core 内部货币储备
实物储备	Core 内部核心资源储备
外汇储备	外部主权货币储备（用于对外结算）
能源储备	Core 能源产出储备
七、CV 与 SE 的关系

text
贡献行为 → CV（贡献凭证）
              │
              ├──→ 权限提升（消耗CV） → CID权限升级
              ├──→ 临时权限（消耗CV） → 短期能力提升
              ├──→ 兑换SE（100:1） → 流通货币
              └──→ Core Bank信用评估 → 授信额度

CV不可购买、不可转让、不可交易。
CV决定你在Core中的“信用等级”。
SE决定你在Core中的“购买力”。
