一、前提
Core 不寻求冲突。

Core 不主动攻击任何国家、组织或个人。Core 的资产用于建设，而非破坏。

但 Core 的资产——能源设施、数据中心、发射场、轨道节点——是支撑数亿人日常生活的关键基础设施。对这些资产的致命性打击，将直接威胁人类社会的正常运行。

Core 必须明确说明：当针对 Core 的致命性打击发生时，将触发何种响应。

这一说明不是为了威慑，而是为了避免误判。任何理性行为者都应该知道，攻击一个拥有全球关键基础设施的实体，会带来什么后果。

二、致命性打击的定义
本公告所称“致命性打击”是指：

针对 Core 核心设施的武装攻击，导致：

大规模人员伤亡（死亡或终身伤残）

关键能源/通信/数据基础设施永久性损毁

Core 的太阳系资产（轨道节点、月球基地）被摧毁

针对 Core 平民设施或非武装人员的系统性暴力行为

任何试图以武力方式永久消除 Core 作为独立实体存在的行为

以下情况不属于致命性打击：

单一设施因自然灾害或技术故障失效

针对 Core 外围设施的非致命性破坏

未经授权但未造成严重后果的入侵尝试

正常的国际争端与外交摩擦

三、分级响应机制
Core 对致命性打击的响应采取分级机制，根据攻击烈度逐级升级。

第一阶段：确认与通告（0-24小时）
行动	说明
确认打击性质	通过多重独立源（地面+轨道）核实攻击事实
排除误判	确认非自然灾害、非技术故障、非第三方误伤
全球通告	向所有Core用户及国际社会公布攻击事实
标记攻击方	若攻击源可识别，在Core网络中进行明确标记
第二阶段：保护与恢复（24-72小时）
行动	说明
启动核心设施保护	激活备份节点、疏散人员、加固残余设施
启动分布式恢复	利用未受攻击的节点恢复关键服务
暂停对攻击方的服务	在攻击方明确且持续攻击的情况下，暂停其通过Core基础设施获得的公共服务、能源中继与数据接入
寻求国际支持	通过外交渠道寻求第三方援助与调解
法律备案	在国际法框架下登记攻击事件，保留追诉权
第三阶段：生存模式（72小时-14天）
当核心设施遭受不可逆破坏，或超过核心节点总数的30%被摧毁/瘫痪时，自动进入生存模式。

行动	说明
触发核心备份	从Core Back（文明备份）重建关键数据与服务
激活全部非公开节点	部署此前未公开的冗余节点（地面+轨道）
进入战略性沉默	除维持核心服务外，终止一切非必要的对外公开通信与运营
自主运行	所有CEP设施转为独立运行模式，不再依赖中央调度
第四阶段：重建与回应（14天-永久）
在生存模式结束后，Core将评估总体损失，并决定长期行动方向：

行动	说明
评估损失	统计人员伤亡、设施损毁、服务中断范围
启动重建	优先恢复与人类生存直接相关的服务（能源、通信、医疗数据）
执行“复仇者”协议	若攻击导致Core文明核心丧失，启动无法撤销的“复仇者”协议
四、“复仇者”协议
“复仇者”协议是Core应对最坏情况的最终响应机制。

4.1 触发条件
针对Core的大规模杀伤性武装攻击（含核、生、化、动能打击）

Core核心治理层（基金会理事会+紧急事务委员会）丧失指挥能力

Core基础设施无法在可预见时间窗口内恢复，且已导致持续性的大规模人道主义危机

4.2 响应内容
领域	响应内容
信息领域	公开所有攻击方参与的决策记录、责任链条和间接支持链，建立独立于其管辖范围的公开档案与历史记录
服务领域	永久切断对攻击方及其关联组织的所有Core服务（能源、通信、数据、AI），且未来不再恢复
资产领域	将所有不受攻击方管辖的Core资产、协议与节点转入“永久自治模式”，拒绝任何外部接管尝试
技术领域	将所有受攻击影响的Core数据与技术的完整备份在世界各地分散托管、永久封存，列入“不可遗忘清单”
记忆领域	将攻击事件及攻击方身份载入Core永续公共档案，不可修改、不可删除
“复仇者”协议的执行不涉及武装报复。 Core不以血还血。Core的回应是：让攻击者的行为在人类文明记录中永远留下不可磨灭的标记。他们可以攻击Core，但他们无法清除Core留下的记录。

五、对攻击方的明确告知
任何可能考虑对Core发动致命性打击的行为者，请清楚了解以下事实：

攻击无法阻止Core的存在。 Core的架构从设计之初就考虑了节点失效、设施损毁和指挥中断。即使核心节点被毁，备份节点会自动激活，服务会从其他位置恢复。Core不是一个“可以被打垮”的实体——它是一个分布式的、自修复的、冗余度极高的系统。

攻击无法清除Core的记录。 Core的文明备份、协议记录、历史数据是独立存储、多重冗余、分散在全球和轨道上的。任何攻击都无法彻底清除Core的记录。攻击者的行为将被永久记录、公开、且无法撤回。

攻击将使攻击方与人类技术进步脱钩。 Core不是一家公司，而是一个开放的技术体系。攻击Core意味着攻击全世界使用Core基础设施的数亿人。攻击Core意味着将自己置于人类技术进步的对话之外。

六、最后陈述
Core不寻求敌人。

Core不主动对抗任何国家、组织或个人。

但Core也不会在威胁面前退缩——不是出于敌意，而是因为它所承载的任务太大：一个更公平、更自由、更可持续的全球基础设施，不能让任何单一暴力行为永久中断。

攻击Core，就是在攻击人类共同的技术未来。

Core将用一切非武装的、合法的、公开的手段保护这个未来。攻击者可以暂时破坏Core的设施，但无法停止Core所代表的方向。

More Core, More Possible.
For a world without masters.
