一、概述

1.1 定义

Core 教育体系是 Core 生态内知识传承、能力培养与人才发展的基础设施。它不以年龄或学龄划分阶段，而是以个人能力与完成度为依据，采用实践主导、模块化学习的模式，确保 Core 在代际更替中保持连续性与活力。

1.2 设计原则

原则	说明
能力本位	进度由个人能力决定，而非固定学制
实践主导	75% 实践评估 + 25% 理论评估
模块化	自定进度，不分学期，可随时进入
开放通路	完成学段后可选择继续深造或进入社会
必修+选修	统一必修 + 多方向选修，兼顾一致性与多样性
二、核心原则

2.1 不采用学年制

Core 教育体系不设固定年限。学习进度由个人的能力增长决定，而非时间推移。

没有“年级”概念
没有固定的学期起止时间
没有统一的教学日历
每个学习者根据自己的进度推进
2.2 能力本位评估

评估依据个人实际表现，而非时间投入或课程完成量。

评估维度	权重	说明
个人实践	75%	实际项目、任务完成、动手操作
理论学习	25%	知识掌握、概念理解、理论应用
评估以能力评估为主，学习者需证明其具备相应能力才能进入下一阶段。

2.3 学习与工作的衔接

完成某一学段后，学习者拥有两种选择：

选择	说明
继续深造	进入下一学段，进一步拓展能力
进入社会	依据该学段对应的能力水平进入 Core 劳动体系
这一选择是双向的：学习者自主决定方向，但 Core 劳动体系也会根据其能力评估结果提供对应的岗位通道。

2.4 与 CI 培训机制的衔接

Core 教育体系与 CI（Core Insurance）培训机制构成递进关系：

机制	定位	说明
CI 培训	基础能力重建	针对低CV低SE人群，提供基础技能培训，使其具备进入教育体系的能力
Core 教育体系	系统能力发展	面向所有成员，提供从启蒙到专业的完整教育通道
CI 培训是“接住人，送出去”，Core 教育体系是“让人走得更远”。两者在功能上互补，不重叠。

三、学段结构

3.1 启蒙阶段（基础学段）

面向所有新进入 Core 生态的成员。完成此阶段后方可进入后续学段。

属性	说明
模式	必修
内容	Core 基本原则、Core 体系概述、基础工具使用、身份认知
评估	实践任务 + 基础理论考核
完成标准	具备 Core 生态的基本认知与操作能力
后续路径	可进入通用阶段，或进入 Core 基础劳动岗位
3.2 通用阶段（基础学段）

面向完成启蒙阶段后希望进一步发展的成员，提供 Core 体系的基础知识。

属性	说明
模式	必修
内容	Core 体系基础：工业基础、能源基础、通信基础、经济与社会基础、安全基础
评估	实践项目 + 理论理解评估
完成标准	具备 Core 体系的基础认知与跨领域协作能力
后续路径	可进入专业阶段，或进入 Core 通用岗位
3.3 专业阶段

面向完成通用阶段后选择特定方向的成员。

属性	说明
模式	选修（选择至少一个专业方向）
方向	CISH/CISL/CISC 工业制造 / CAD 航空航天与轨道运维 / CEC/CEP 能源体系 / CSN/CS 通信与网络 / Core Security 安全与防御 / Core Medicine 医疗与健康 / Core Governance 治理与公共事务 / Core Science 基础科学
评估	专业实践项目 + 专业理论评估
完成标准	具备对应专业方向的实际工作能力
后续路径	进入对应部门工作，或继续拓展其他专业方向
3.4 终身阶段

面向所有阶段的成员开放，不设前置条件，可在任何时间进入。

属性	说明
模式	选修
内容	持续更新课程、跨领域拓展、新技术学习、高级研讨、导师指导
评估	持续参与记录 + 实际贡献评估
完成标准	不设“毕业”概念，以持续参与为主要状态
四、课程模式

4.1 必修课程

固定的核心课程体系，所有学习者必须完成。

课程类别	内容
Core 理念与历史	Core 的起源、核心原则、独立宣言、转型原则、熔断与授权
Core 体系通识	Core 的组织结构、主要部门、运行方式
Core 公民教育	权利与义务、社会契约、参与方式、责任意识
必修课程确保每一位 Core 成员对 Core 本身有统一的基本认知。

4.2 选修课程

面向特定方向的深度课程，学习者根据自身发展路径选择。

方向类别	典型课程
工业制造	CISC 资源采集、CISH 重型制造、CISL 微观制造、质量控制
航空航天	CAD 轨道运营、CSS/CIN 维护、Core Rocket 发射与回收
能源体系	CEC 调度、CEP 发电、核聚变运维、轨道太阳能管理
通信网络	CSN 组网、星间链路维护、深空通信协议
安全与防御	防御体系、武装力量、危机响应、情报分析基础
医疗与健康	Free Medical 体系、太空医疗、心理支持
治理与公共事务	Core 治理结构、经济管理、社会服务、教育体系
基础科学	空间科学、能源科学、材料科学、计算科学
五、评估与认证

5.1 综合评分

每个学段的完成度由综合评分决定：

text
综合评分 = 实践评分 × 75% + 理论评分 × 25%
评分等级	说明
通过	综合评分达到标准线，可进入下一阶段或进入社会
未通过	综合评分未达标，需要补足相应部分后重新评估
优秀	综合评分显著超出标准线，可获得额外 CV 奖励
5.2 能力认证

完成学段后，学习者获得对应的能力认证，与 CID 绑定。

认证	对应学段	效力
Core 基础认证	启蒙阶段	证明具备 Core 基础认知与操作能力
Core 通用认证	通用阶段	证明具备 Core 体系基础能力
Core 专业认证	专业阶段	证明具备特定专业方向的实际工作能力
Core 持续认证	终身阶段	证明持续参与与更新状态
能力认证作为 CID 的附加标记，在 Core 体系内有效，且可被 Core 劳动体系直接识别和对接。

5.3 能力评估与岗位对接

完成学段后，学习者的能力认证与 Core 劳动体系岗位需求自动对接，对接过程不需要经过额外审核或考核环节。

学段完成	对接岗位类型
启蒙阶段	Core 基础岗位（需配合岗位培训）
通用阶段	Core 通用岗位
专业阶段	对应部门专业岗位
六、学习与贡献的转化

转化方向	说明
完成课程 → CV	完成教育模块可获得 CV 奖励
完成学段 → CV	完成整个学段可获得额外 CV 奖励
进入岗位 → CV	完成学段后进入实际岗位可获得额外 CV 奖励
参与教育建设 → CV	设计课程、指导学员、编写教材可作为贡献行为计入 CV
学习成果 → 权限	完成学段的能力认证可提升 CID 权限等级
七、教育治理

7.1 Core Education Council（CECedu）

Core 教育体系由 Core Education Council（CECedu）统一管理，直接向 Core 基金会汇报。

职责	说明
课程制定	与 CRADT 及各部门联合制定课程内容
评估标准	制定各学段的能力评估标准
导师管理	管理与调配教育体系中的导师资源
持续更新	定期更新课程内容，与 Core 实际需求保持同步
与劳动体系对接	确保教育输出与 Core 劳动体系需求匹配
7.2 导师制度

导师从 Core 各部门在职人员中轮换调用
担任导师可作为贡献行为计入 CV
导师需完成 Core 教育体系认可的导师培训
导师的选拔与轮换周期由 CECedu 统一管理，确保知识传递的连续性和质量
7.3 教育资源的开放

所有教育资料向全体 Core 成员开放
任何人都可以自学课程内容并申请能力评估
不强制进入教育体系，但进入需要相应能力的岗位时必须通过对应能力认证
八、教育在 Core 中的地位

教育体系是 Core 基础设施的组成部分，与工业、能源、通信、治理处于同等地位。

维度	说明
代际连续性	教育是 Core 跨越代际的唯一途径
社会流动性	教育提供不受出身限制的上升通道
知识传承	教育确保 Core 的知识体系不被遗忘
能力储备	教育确保持续有人才进入 Core 的各个部门
如果 Core 需要持续存在超过一代人的时间，教育体系就是必须建立的机制之一。它不是可选功能，而是必要结构。
