一、目的与范围
本文件规定 Core 在遭受武装攻击时，进行自卫性武装反击的正当性条件、授权路径、行动界限与事后责任。其目的是：

明确 Core 行使自卫权的合法边界

确保武装反击行动始终服从于防御目的

防止武装力量被滥用或超出必要限度

本反击案仅适用于 Core 遭受明确、持续、非法的武装攻击，且非武装手段已被证明不足以阻止威胁的情况。

二、正当性前提
Core 武装反击的正当性建立在以下全部条件同时满足的基础上：

攻击已经发生：存在客观、可核实的武装攻击证据

攻击对象明确：攻击方身份可辨识，攻击目标属于 Core 合法资产或人员

攻击仍然持续或即将再次发生：反击不是为了报复，而是为了阻止当前或迫在眉睫的威胁

非武装手段已穷尽：外交抗议、网络防御、法律途径均无法阻止攻击

反击符合比例原则：反击力度不超过“阻止攻击”所需的最低限度

反击目标严格限于军事目标：不针对平民、民用设施或非武装人员

上述任何一项条件不满足时，Core 武装力量不得实施反击。

三、反击授权链条
武装反击的授权必须经过以下层级，逐级确认：

层级	授权主体	权限范围
第一层	Core 紧急事务委员会	确认攻击事实，评估威胁等级，决定是否启动反击预案
第二层	Core 基金会理事会	批准大规模或跨区域反击行动（需2/3多数）
第三层	现场战术指挥官	在紧急情况下进行局部、即时、有限的自卫反击，但在24小时内报请紧急事务委员会追认
所有授权记录须完整保存，纳入事后审查。

四、反击行动界限
4.1 目标选择
反击目标仅限于正在实施攻击的军事单位、发射平台、指挥节点及直接支持其攻击行动的设施

禁止攻击：

平民居民区、学校、医院

文化遗址、宗教场所

核电站、大坝等可能造成次生灾难的目标

非武装的辅助设施

攻击方国内的民用基础设施

4.2 武力等级
等级	适用情况	允许手段
一级（最低）	攻击为轻武器、无人机投弹	精确打击攻击源，使用非致命或低附带损伤武器
二级（中等）	攻击包含重武器、火炮、导弹	对攻击源进行直接摧毁，可动用防御性精确制导武器
三级（最高）	攻击威胁 Core 生存，或包含大规模杀伤性武器	在严格限制下，可对攻击方的军事指挥与发射设施进行精准打击，但必须确保不产生不可逆生态灾害
任何情况下，不得使用核武器、生物武器、化学武器，不得对城市区域进行轰炸，不得对非军事目标进行打击。

4.3 反击时限
反击行动应在“攻击持续期间”或“可信的再攻击威胁存在期间”进行

攻击停止后，反击立即终止

不得进行超过24小时的连续反击行动，除非新的攻击已经发生

五、反冲击原则
Core 的反击不以“报复”为目的，不以“摧毁对方”为目标，不以“确保对方无力再次攻击”为长期目标。其唯一目的是：

使正在进行的攻击停止

消除攻击方的继续攻击能力

为 Core 自身的防御恢复创造条件

反击行动一旦达成上述目的，应立即停止，并主动通报停火。 Core 不追求“消灭敌人”，只追求“终止威胁”。

六、事后审查与问责
每次武装反击行动结束后，必须在 72 小时内启动独立审查

审查内容包括：行动合法性、目标选择合理性、武力使用比例性、附带损伤情况

审查委员会由 CPT 指定独立成员组成，不包括参与行动的人员或直接利益相关方

若审查发现行动违反本反击案任何条款，相关责任人应被追责（包括免职、剔除共建资格等）

审查结果应向全体共建者公开（涉密技术细节除外）

七、与国家自卫权的兼容性
Core 不是国家，但 Core 拥有实体资产和受保护人员，其自卫权源于国际习惯法中的“自卫”原则。Core 的反击行动不视为对任何国家主权的侵犯，而是对自身合法存在的防卫。

Core 在行使自卫权时，承诺：

尽最大努力减少附带损伤

不破坏当地民众的正常生活秩序

反击开始前尽可能发出警告（除非警告会使防御失效）

反击结束后主动配合第三方人道主义评估

八、最后声明
Core 武装反击不是战争行为，而是防御行为的延伸。

Core 不希望使用武装反击。武装反击意味着 Core 的和平努力已经失败，意味着物理世界中最糟糕的部分已经降临到 Core 的头上。

但如果必须使用，Core 将以最克制、最精准、最短暂的方式进行，并在结束后立即回归和平轨道。

我们不是为了伤害谁而存在，但也不会因为害怕被伤害而放弃存在。
